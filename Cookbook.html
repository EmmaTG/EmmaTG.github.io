My Cookbook

Technology used

  • Java
  • MySQL
  • Angular
  • Spring Boot

This project was inspired by my sister's frustration of not knowing what to cook for dinner given the ingredients she already had at home. This app allows one to search for recipes given certain ingredients. One is also able to add recipes and specify the main ingredients in this recipe. The recipes can then be sorted by many different categories. This app has a Springboot/Java backend and an angular frontend with stylings by primeNG. It uses mySQL for the database. Visit my gitHub repository for instructions on how to run you application locally including mySQL creation scripts.
